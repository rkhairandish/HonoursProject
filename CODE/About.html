Instructions To Use Software/Device and Save Files As CSV

1 - Put GeneActiv Wrist Accelerometer on desired wrist

2 - Start by Pressing Button on GeneActiv (long green flash)

3 - Punch away

4 - Stop by Pressing Button on GeneActiv again (long red flash)

5 - Plug GeneActiv into PC and Extract/Download CSV file from GeneActiv Software

6 - Save file (something like BTT3.csv recommended) into given location (HonoursProject/CODE/data/..)

7 - Go to website (https://zeno.computing.dundee.ac.uk/2017-projects/boxing/)

8 - View Timeline Graph, Count Punches ETC
